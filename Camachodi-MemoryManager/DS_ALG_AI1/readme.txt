Instructions

1.- Open a console in directory DS_ALG_AI1.
2.- Modify "./tools/genie.lua" file to add new files (headers and source and test files) for each new project at the end of the file.
3.- Execute "./tools/compile_vs2017.bat" or "./tools/compile_vs2019.bat", depending on the version you are using.
4.- In the directory "./build" has been created a solution for "Visual Studio". This is a global solutions that include every project (in a separate way) we will work in class.
5.- Double click on the solution and open the Visual Studio IDE.
6.- Select the project you want to work.
7.- May the force be with you.
